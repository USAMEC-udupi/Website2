Image Configuration Usage Example

How to Use the Image Configuration

1. Import the image configuration:

import { images, getImage } from './src/images.js';

2. Use images directly:

<img src="${images.hero.main}" alt="Hero Image">

3. Use the helper function:

<img src="${getImage('hero', 'main')}" alt="Hero Image">

4. Update an image dynamically:

updateImage('hero', 'main', 'new-image-url.jpg');
